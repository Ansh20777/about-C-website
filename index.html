C is a general-purpose programming language. It was created in the 1970s by Dennis Ritchie and remains widely used and influential.

C is a powerful, low-level procedural programming language used to build operating systems, embedded systems, and more.

Features of C

  • Simple and structured programming language
  • Fast execution and high performance
  • Low-level access using pointers
  • Portable across different platforms

Steps to Compile a C Program

  1. Write the source code in a .c file
  2. Save the file using a text editor or IDE
  3. Compile the code using a C compiler (gcc)
  4. Run the generated executable file

Definitions

Compiler
A program that converts C source code into machine code
Pointer
A variable that stores the memory address of another variable
Variable
A named memory location used to store data values
